# proyectos-tripleten
Portfolio de proyectos realizados durante el bootcamp de ciencia de datos

## Proyecto 5
Análisis de datos de telecomunicaciones para la compañía Megaline. El proyecto compara los ingresos generados por dos planes tarifarios (Surf y Ultimate), analiza el comportamiento de los usuarios en términos de llamadas, mensajes e internet, y evalúa diferencias regionales, particularmente entre Nueva York-Nueva Jersey y otras ciudades. Se incluyen pruebas de hipótesis estadísticas para validar las diferencias observadas.

## Proyecto 6
Análisis de ventas de videojuegos históricos para identificar patrones que determinan el éxito de un juego. El estudio examina datos de ventas por plataforma, género y región, analiza las calificaciones de críticos y usuarios, y proporciona recomendaciones estratégicas para el lanzamiento de juegos en 2017, enfocándose en plataformas emergentes y géneros populares.
