# proyectos-tripleten
Portfolio de proyectos realizados durante el bootcamp de ciencia de datos

## Proyecto 1
Primera fase del análisis de datos para Store 1. Enfocado en la limpieza y preparación de datos de clientes, eliminando duplicados y asegurando consistencia en el registro de nombres y apellidos.

## Proyecto 2
Segunda fase del análisis de datos para Store 1. Aplicación de técnicas avanzadas para procesar y analizar datos de clientes y compras, con el objetivo de identificar clientes leales, analizar ingresos por categoría y encontrar oportunidades de marketing segmentado.

## Proyecto 3
Análisis de datos musicales para examinar hábitos de escucha. El proyecto explora un conjunto de datos de música para identificar patrones en el comportamiento de los usuarios, géneros populares y tendencias de reproducción.

## Proyecto 4
Análisis de datos de Instacart, una plataforma de entrega de comestibles. El estudio examina pedidos, productos y hábitos de compra para identificar patrones en el comportamiento de los consumidores y oportunidades de mejora en la logística.

## Proyecto 5
Análisis de datos de telecomunicaciones para la compañía Megaline. El proyecto compara los ingresos generados por dos planes tarifarios (Surf y Ultimate), analiza el comportamiento de los usuarios en términos de llamadas, mensajes e internet, y evalúa diferencias regionales, particularmente entre Nueva York-Nueva Jersey y otras ciudades. Se incluyen pruebas de hipótesis estadísticas para validar las diferencias observadas.

## Proyecto 6
Análisis de ventas de videojuegos históricos para identificar patrones que determinan el éxito de un juego. El estudio examina datos de ventas por plataforma, género y región, analiza las calificaciones de críticos y usuarios, y proporciona recomendaciones estratégicas para el lanzamiento de juegos en 2017, enfocándose en plataformas emergentes y géneros populares.

## Proyecto 7
Aplicación web para el análisis de datos de vehículos usados en EE.UU. Desarrollada con Streamlit, permite explorar y visualizar datos de anuncios de venta de autos, facilitando el análisis de precios, características y tendencias del mercado.
